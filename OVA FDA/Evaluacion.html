Evaluacion para cada deporte

Tambien es importante evaluar el desempeño en los deportes urbanos. Aquí hay algunas actividades relacionadas con los diferentes deportes en las cuales se haran evaluaciones sencillas a partir de cada deporte:

  • Para resolver estas evaluaciones, antes, se debió leer acerca de cada deporte, que actividades se realizan para cada uno y que medidas de seguridad hay para poder ejecutar cada uno de la nera correcta.
  • Estas evaluaciones no definen tu rendimiento en cada deporte, pero si te ayudaran a reforzar tus conocimientos en cada uno.
  • Para cada deporte hay una evaluacion correspondiente, lee cuidadosamente y si con anticipacion leiste sobre cada deporte, se te hara mas facil responder.
